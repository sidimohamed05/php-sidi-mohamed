Comment exécuter réellement PHP ?

GitHub Pages ne lance pas PHP. Cette plateforme fonctionne comme support statique. Pour exécuter les fichiers PHP, utilisez un serveur local.

Méthode

cd votre_dossier
php -S localhost:8000

Puis ouvrez http://localhost:8000. Vous pouvez aussi utiliser XAMPP, Laragon ou WAMP.
